# Brief Description

# 🔐 SecureVault

- SecureVault is a security-focused web application for securely storing sensitive information and monitoring account activity in real time.
- It combines secure storage, authentication, activity tracking, session management, and basic threat detection into a single platform.

## ✨ Features

- 🔑 **Secure Vault** – Store passwords, API keys, and private notes with encryption.
- 🔐 **Authentication** – Secure registration, login, JWT authentication, and bcrypt password hashing.
- 📊 **Activity Logging** – Track logins, logouts, and secret management actions.
- 🧑‍💻 **Session Management** – View active sessions and remotely log out devices.
- 🚨 **Threat Detection** – Detect suspicious login attempts and unusual account activity.
- 📈 **Security Dashboard** – Monitor recent activity, login history, alerts, and risk indicators.

## 🎯 Goal

Build an industry-level security application that demonstrates secure authentication, encryption, activity auditing, session management, and threat detection concepts.
